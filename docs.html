Get up and running.

Installation, your first scene, mapping a controller, and shipping a recording. More guides land as 1.0 ships.

Installing

Download the installer from the Download page, run it, and follow the prompts. Fluence drops itself into C:\Program Files\Fluence by default. Project files and recordings live alongside your user folder.

Your first scene

  1. Open Fluence. The composition starts empty with three layers.
  2. Open Browser → Sources and double-click Spiral to drop it on Layer 3, slot 1.
  3. Press L — the slot triggers and the spiral renders into the viewport.
  4. Drag Plasma onto Layer 2, slot 1, and trigger it. Layers composite top-down.
  5. Use the deck transport bar to set BPM and start the master clock.

Mapping a controller

Open Bindings → Edit MIDI (or Edit Keyboard / Edit OSC). Every bindable control gets a coloured overlay. Click a slot, fader, or parameter, then move a knob or press a key — Fluence captures the binding. Bindings show up in the Bindings panel where you can adjust min/max ranges or unmap them. Save the project; the bindings travel with it.

One source can fan out to several targets. To trigger a whole scene from one key, just bind that key to each clip you want firing.

Recording your set

The transport bar has a record button. Hit it once to start, again to stop — files land in the Recordings/ folder next to the app. Codec, frame rate, and resolution are configurable in Output → Recording.

Useful shortcuts

  • Space — play/pause master clock
  • T — tap BPM
  • Ctrl+S — save project
  • Ctrl+Z / Ctrl+Y — undo / redo
  • F11 — toggle full-screen output
