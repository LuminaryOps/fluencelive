What's shipping.

Release notes for every Fluence build, newest first. Bug fixes, new features, and the occasional breaking change.

Beta — current

In active development. Polish, stability, and the run-up to 1.0.

  • Unified deck and arrangement transport bars
  • Bindings panel with Keyboard / MIDI tabs and inline range editing
  • OSC server with address parsing and dispatch
  • MIDI input thread, bind capture, and dispatch
  • Toolbar transport mappings and per-layer controls
  • Preferences dialog with MIDI and OSC tabs

Coming soon

  • Group, arrangement, and parameter mapping targets
  • SDF modeler — mesh import and additional primitives
  • Plugin system — VSCode-style architecture
  • Style transfer — SDXL + Temporal pipeline

Detailed release notes start with the 1.0 cut. Subscribe to updates by emailing hello@fluencelive.com.
