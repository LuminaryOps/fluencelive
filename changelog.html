Changelog

What's shipping.

Release notes for every Fluence version.

2.0.0

The biggest release since launch: a new rendering engine, a second platform, and a control surface for scripts and AI.

  • New renderer. The entire engine moved from OpenGL to wgpu — Metal on macOS, Vulkan on Windows. Modern GPU backends and a foundation for what's next.
  • Fluence on macOS. Native Apple Silicon build, signed and notarized. Full feature set: Syphon in place of Spout, VideoToolbox-accelerated decode and H.264 encode, AVFoundation capture, ScreenCaptureKit audio, NDI.
  • Control API & MCP. A built-in local automation API, off by default, plus a bundled MCP bridge — AI agents and scripts can trigger clips, edit parameters, set up audio-reactive automation, build arrangements, and screenshot the composition.
  • NDI, in and out. A new NDI clip type receives network video sources, and Fluence's output can publish as an NDI sender. Both platforms.
  • Encoders. Hap export and recording on macOS, DXV3 encoding parallelized for real-time recording at full frame rate, hardware H.264 on macOS.
  • Fixes. DXV3 recordings with audio no longer fail at finalize; exports with audio from AVI/WAV-family sources no longer fail; large Control API replies no longer truncate; macOS keyboard handling corrected in web clips and the deck.

1.0

Initial Windows release: the full composition engine — 15 clip types, layers and groups, effects at four scopes, keyboard/MIDI/OSC mapping, timeline and FFT automation, arrangement, projection mapping with edge blending and warping, live recording, and twelve export codecs.
