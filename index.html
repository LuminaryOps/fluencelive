What's inside

One app, every layer of a live rig.

Composition, mapping, mixing, projection, recording. All built as one system.

15 clip types

Shaders, video, audio, images, live web pages, capture cards, screen and window grabs, generators. Anything that renders is fair game.

Map anything

Bind any control to a key, a MIDI knob, or an OSC address. One capture flow across all three. One source can fan out to many targets.

Audio-reactive

Drive any parameter from any frequency band. Pick the range, set min and max, draw an envelope curve. From the room or an external input.

Layered composition

Stack layers, fill slots with clips, group, route. Switch between deck-style triggering and an arrangement timeline you can scrub.

Output anywhere

Multiple displays, projection mapping with slices and warping, edge blending across projectors, Spout in and out for cross-app rigs.

Record & export

Live record your set with bar-aware start and stop triggers. Export to H.264, ProRes, HAP, DXV3, PNG, and more. Twelve codecs in total.

In motion

Made to perform.

Composition

Compose like a deck.
Perform like an arrangement.

A layered deck for live triggering. An arrangement timeline for prepared sets. Both views, same composition. Stack layers, group them, blend across 27 modes, transition with cuts, fades, wipes, or pushes.

Sources

A studio's worth of generators, built in.

ISF and GLSL shaders. Multi-pass Shadertoy import. A point-and-click 3D modeler that generates raymarched GLSL on the fly. Deep-zoom Mandelbrot. Animated text with per-character stagger. Drop your own shader files into a folder and they show up in the browser.

Hardware control

One mapping system. Three protocols.

Keyboard, MIDI, OSC. Captured the same way, stored the same way. Click a control, press a key or move a knob, done. One source can fire a whole scene. Bindings save with the project so the next show starts where you left off.
