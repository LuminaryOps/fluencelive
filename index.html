What's inside

A complete real-time studio

Every layer of a live visual rig — composition, mixing, mapping, recording — in one tool.

Built for 60fps

OpenGL render pipeline, audio-thread isolation, GPU resource discipline. Stays smooth when the room is full.

Shader-native

GLSL and ISF sources load at runtime. Hot-edit a shader, see the result on stage in milliseconds.

Mappable everything

Bind any control to keyboard, MIDI, or OSC. Capture in two clicks; one input can fan out to many targets.

Audio-reactive

Beat detection, FFT, and loopback capture flow into any uniform or parameter. Visuals follow the room.

Layered composition

Stack clips into layers, blend with familiar modes, group and route. Resolume-style flow, modern internals.

Record & output

Spyder-grade screen capture, multi-output routing, and built-in recording — straight to disk, ready to ship.

In motion

Made to perform

Compose like a DAW.
Perform like an instrument.

Layers, slots, columns, and an arrangement timeline you can scrub. Build a set in advance, then trigger it the way you want it played — knobs, pads, keys, or all three at once.

SDF modeling, fractals, and ISF — out of the box.

Built-in raymarched 3D, dynamic lighting, fractal zoom, particle systems, and the entire ISF library. Drop a shader file in, it shows up in the browser. No plugin chain to manage.

One mapping system. Three protocols.

Keyboard, MIDI, OSC — captured the same way, stored the same way. Bind to clip triggers, layer faders, BPM tap, effect parameters. Save it with the project; recall it on the next show.
